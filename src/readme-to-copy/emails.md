# At Rules

## Evaluation Criteria

1. Make it Readable.
2. Good Practices.
3. Use HTML5.
4. Structured Naming Convention.
5. Responsive code is not required.
6. Use correct Cascade code.
7. Use correct Inheritance.
8. Goal: The practice should look as the required design.
